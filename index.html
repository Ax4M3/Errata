KartoLibr - Générateur ONIX 3.0

KartoLibr

Remplissez une fiche, téléversez ou glissez-déposez un XML existant. Tout reste dans votre navigateur.

À lire avant usage professionnel : ce générateur produit un squelette ONIX 3.0 structurellement correct pour un cas simple. Il ne fait pas de validation XSD complète et la liste de classification ci-dessous est volontairement réduite. Vérifiez les codes auprès des listes officielles EDItEUR avant envoi à un distributeur. Vérification : Vous pouvez vérifier le fichier XML généré grâce à l'outil XML validation
